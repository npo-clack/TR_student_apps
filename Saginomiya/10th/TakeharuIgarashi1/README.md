# 地理クイズゲーム (Quiz-AI)

このプロジェクトは、HTML / CSS / JavaScript で作ったシンプルな5択クイズゲームです。
日本の都道府県の県庁所在地問題と、世界の国の首都問題の2種類を学べます。

## 目的
- 都道府県の県庁所在地や国の首都を学ぶための簡単な学習用ウェブアプリ。
- 高校生でも読めるように、シンプルな実装と分かりやすいコメントを心がけています。

## ファイル構成
- `index.html`  - アプリの HTML。開始画面 / クイズ画面 / 結果画面 を含みます。
- `styles.css`  - 見た目（レイアウト・色）のスタイル定義。
- `questions.js` - 問題データ。都道府県（47件）と世界の首都（約50件）を配列で保持。
- `script.js`   - ゲームのロジック（問題のシャッフル、選択肢生成、採点、結果表示など）。
- `README.md`   - （このファイル）プロジェクトの説明と実行手順。

## 特徴（現在実装されている主な機能）
- クイズの種類を選択（都道府県 / 世界の首都）
- 五択（正解1つ + 誤答4つ）で出題
- 最大20問をランダムに出題（データが20未満ならデータ数分）
- 毎回問題がシャッフルされ、同じ順番にならない
- 都道府県問題では、正答を見破りにくくするため問題の県名をもとにした偽の「〜市」を1つ誤答候補として混ぜる（例: 「宮城市」）
- 結果画面で何問正解か表示、さらに各問題ごとに「あなたの答え」と「正解」を表示（◯/✕付き）

## 使い方（すぐ遊ぶ）
1. 作業ディレクトリで `index.html` をダブルクリックしてブラウザで開いてください。ローカルでそのまま動作します。

2. （オプション）簡易サーバーで確認したい場合（推奨: 一部ブラウザはローカルファイルでリソース制限することがあるため）PowerShell で次のように起動できます（Python が必要）:

```powershell
# ワークスペースのルートで実行
python -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

## 問題データの編集・追加
- 問題はすべて `questions.js` に書かれています。形式は以下のとおり:

```javascript
{ question: "〜〜の県庁所在地は？", correct: "正解の市名", wrong: ["誤答1","誤答2","誤答3","誤答4"] }
```

- 都道府県は `prefectureData`、世界の首都は `capitalData` に配列でまとめられています。
- 問題を追加するときは上記の形式に従って要素を追加してください。

## カスタマイズ案（今後の拡張）
- 誤答のクオリティを上げる（地理的近さや規模で妥当な選択肢を優先）
- 問題数を設定できる UI を追加（20問以外を選べる）
- タイマーや問題ごとの解説を追加
- ローカルに学習履歴を保存して苦手な問題を分析する（localStorage を使用）

## 既知の制限・注意点
- 問題データはクライアント側（ブラウザ）に置いてあるため、デベロッパーツールで答えが見えてしまいます。サーバー側で管理するとより安全です。
- 偽の「〜市」は自動生成されるため実在しない名称が表示されます。これは意図的な挙動です（問題を単純に見破られないため）。

## 開発メモ（コードの見どころ）
- `script.js` の `buildChoices()` は、都道府県問題で偽の「〜市」を1つ混ぜるロジックを持っています。
- `script.js` の `userAnswers` 配列は、ユーザーが選んだ答えを記録し、結果画面で表示するために使います。
